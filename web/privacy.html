thingino Firmware Builder, last updated 30 June 2026

thingino Firmware Builder is a free community tool for compiling thingino camera firmware. We collect as little as possible: no accounts, no cookies, no analytics, no third-party trackers.

What we log

  • Your IP address, with each build request, used only to enforce rate limits and stop abuse (e.g. flooding the build queue).
  • An anonymous builder ID: a random value we generate that your browser stores, so we can show you your build's status. It isn't linked to your identity.
  • The camera profile (defconfig) and branch you build, needed to run the build, and kept for basic operational stats.

We do not collect names, emails, passwords, or other personal data.

Builds and downloads are public

Your firmware is compiled on GitHub Actions and published as a public download (a GitHub release asset). While it exists (about 30 minutes, then it's auto-deleted), anyone with the link can fetch it. The images hold no personal data, but the link isn't secret.

Who processes your data

The site is served by GitHub Pages, the build API runs on Cloudflare Workers, and builds run on GitHub Actions. Your requests and IP pass through Cloudflare and GitHub, each with its own privacy policy.

Retention

IP addresses and request logs are kept only briefly, for abuse prevention, and are purged periodically. Built images are removed after ~30 minutes.

Stored in your browser

A few preferences live on your device: language, help-hints on/off, selected branch, and your anonymous build ID. Only the build ID is ever sent to the service.

Changes & contact

We may update this policy; the date above shows the latest version. Questions: open an issue at github.com/thingino/thingino-web-builder.
